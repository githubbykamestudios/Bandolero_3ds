# Bandolero_3ds
# ¿Que es bandolero 3ds?
Bandolero 3ds es una aplicacion para 3ds basada en Notepad 3ds que por el momento solo permite visualizar archivos de texto.
Tendra mejoras con respecto a Notepad 3ds proximamente.
